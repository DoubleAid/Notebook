```
# $ 表示 当前shell进程的 ID
echo $$     -> 输出 29949
```

|  变量   | 含义  |
|  ----  | ----  |
| $0  | 当前脚本的名称 |
| $n  | 传递给脚本或者函数的参数， n是一个数字，表示第几个参数 |
| $#  | 传递给脚本或者函数参数的个数 |
| $*  | 传递给脚本或者函数的所有参数 |
| $@  | 传递给脚本或者函数的所有参数，与 $* 的区别见下面的详解 |
| $?  | 上个命令的推出状态 或者 函数的返回值 |
| $$  | 当前shell进程的ID |
