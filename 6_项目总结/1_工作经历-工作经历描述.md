# 项目介绍

## 中智行

我在中智行是做L4自动驾驶的SLAM的开发，这个SLAM主要的用途：一个是给运行的车辆提供高频的绝对位姿，第二个是将优化后的位姿保留用于制作高精地图

使用的主要传感器有激光雷达，IMU，轮速记和GPS，其中使用IMU做高频预测，其他传感器做低频修正

在预处理阶段主要是通过滤波算法剔除离群干扰点，结合动态点检测剔除动态目标，减少点云数量，提取线/面特征点，通过iKDTree存储点云，使用 FastGICP 通过scan-submap 或者 scan-离线地图 进行配准

里程计部分采用滑动窗口和ESKF配合实现多传感器的融合，使用IMU预积分数据作为高频位姿预测，其他传感器的数据作为低频观测，通过ESKF的误差状态模型，实时修正窗口内的位姿和 IMU 零偏，避免单一传感器漂移导致的里程计失效。

后端融合激光雷达的关键帧配准位姿约束，IMU预积分的位姿约束，GPS定位数据/轮速记历程信息以及闭环结果，进行位姿图优化，优化的结果也会作为一个精准的约束去求证里程计的零偏和位姿增量

## 图达通

我在图达通负责 SLAM 框架的定制化开发与场景化优化，雨天模式专项优化与固态激光雷达回环检测算法实现，均聚焦于提升 SLAM 系统在复杂场景下的鲁棒性与精度

雨天模式，这个功能主要是为了解决雨雾天气下激光雷达会有大量的雨水点和高反膨胀点，会影响感知的物体检测和配准精度，通过噪点比例自动或者手动切换到雨天模式，在预处理阶段会通过统计滤波结合强度标记雨水点，使用衰减矩阵对高强度点周围做强度衰减，衰减后强度低于阈值的标记为膨胀点，最终输出去噪后的干净点云

后端针对针对雨天点云配准残差波动大的问题，动态调整多传感器约束权重（降低激光配准约束权重至原权重的 60%，提升 IMU 预积分约束权重至 40%），并替换鲁棒核函数（从 Huber 核切换为 Cauchy 核），强化对异常残差的抑制能力；

## 大众

我在一汽大众主要负责泊车场景 SLAM 系统开发，重点落地两大核心任务 ——IMU + 轮速计融合 DR（航位推算）开发、雷达 - 相机在线标定工具研发；

IMU + 轮速计融合 DR 开发，为泊车感知模块提供短时高频初始位姿

采用 “IMU 高频预测 + 轮速计低频修正” 的 ESKF 误差状态融合方案，与中智行的里程计核心逻辑一致，但针对泊车低速、频繁启停的特性，优化了 ESKF 的误差状态模型与噪声矩阵参数；

针对传统 velo2cam 标定板静态标定在泊车场景中存在的外参漂移、精度不足问题，开发在线持续标定工具，核心目标是实时输出精准外参与时间偏差，为感知真值地图构建提供可靠的传感器外参基准

+ 前端预处理与特征匹配：
  + 特征提取：图像侧提取 ORB 特征点（兼顾实时性与鲁棒性），点云侧提取角点、平面点等结构化特征（提升配准稳定性）；
  + 初始配准与点对筛选：基于激光雷达与相机的初始外参（静态标定值）和相机内参，先将点云特征点投影至图像平面，随后在该邻域内检索图像 ORB 特征点，计算投影点与 ORB 特征点的像素欧氏距离，保留距离低于阈值（如 3 像素）的有效 2D-2D 匹配对，过滤异常点；
  + 粗配准求解：通过 EPNP 算法求解初始位姿，计算重投影误差，筛选重投影误差≤1.5 像素的高质量点对，为后端优化提供可靠约束。
+ 后端多约束联合优化（解决可观测性与稳定性问题）：
  + 核心约束：以 3D-2D 投影误差作为主约束（保证外参标定精度）；
  + 可观测性优化：补充 IMU 预积分约束与点云配准的运动观测一致性约束，解决泊车场景纯直线匀速运动时外参可观测性差的痛点；
  + 稳定性约束：添加外参平滑约束（限制前后两帧外参差值≤阈值），避免外参突变导致感知抖动；
  + 漂移修正：引入回环检测机制，当检测到场景回环时，通过回环帧的匹配约束修正外参长期漂移；
