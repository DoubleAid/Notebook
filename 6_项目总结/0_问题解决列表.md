## 高精地图生产定位和建图系统开发 

### 项目介绍

主要参与基于传感器融合（激光雷达-IMU-GPS-轮速记）的高精度建图系统的开发，在复杂城市道路场
景下实现<10cm绝对定位精度，并已经应用在相城区80km2的高精地图建图工程中

### 主要工作 

1. 系统滤除激光点云中的动态物体，提取线/面特征点，基于ikdTree实现高效点云管理，使用FastGICP
算法实现帧-局部子图-全局地图的配准
2. 设计和实现误差卡尔曼滤波(ESKF)框架，将IMU角速度/加速度/零偏和激光观测，GPS等纳入状态空间
建模，实现高频(125Hz)位姿预测和低频观测(激光/GPS等)的鲁棒融合，提供高频的绝对位姿输出；
3. 在多传感器融合的时候采用滑动窗口+ESKF配合，处理非及时的传感器测量数据(如激光匹配)，增强
在线定位状态传递和估计的准确性
4. 后端融合激光雷达关键帧约束、IMU预积分、GPS/轮速记观测及闭环检测结果，进行统一的位姿图优
化，构建全局一致性轨迹，建图关键区域绝对精度<5cm;

## 激光雷达SLAM算法栈开发

### 项目介绍

负责固态激光雷达SLAM功能的优化和产品化，将SLAM算法封装为标准化SDK，开发统一API与配置
工具，为主机厂和机器人客户提供即用型SLAM解决方案

### 主要工作

1. 封装 SLAM 核心模块（含预处理、特征提取、配准优化）为可复用 SDK，对外提供统一 API 接口与配置文件，支持中间数据（点云、特征点、位姿）输出及参数动态调优（如特征提取阈值、配准精度等）；
2. 针对雨天场景下激光雷达点云受雨水点、目标膨胀点干扰导致的 SLAM 漂移问题，专项开发雨天模式：通过雨水点与膨胀点筛选机制，结合残差权重动态调整及鲁棒核适配，提升雨天环境下系统运行稳定性；
3. 采用体素网格存储特征点，通过 NDT 算法完成特征点云粗配准，FastGICP 算法实现精配准；后端支持 IMU、GPS、轮速计与回环约束的按需融合，同时输出绝对位姿与相对位姿；
4. 回环检测通过 IKD-Tree 检索最近邻多帧位姿并构建子图，复用 NDT-FastGICP 流程完成位姿配准，最终构建回环残差约束。
