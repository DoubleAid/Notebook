# 常见问题 Q&A

## 1. 李群和李代数的关系

李群与李代数是连续变换群与其局部线性化近似的对应关系，核心可概括为：

+ 李群（如旋转群 SO (3)、欧氏变换群 SE (3)）是光滑的连续群，描述 “可连续变化的变换”（如机器人姿态 / 位姿的连续运动），满足群运算（闭合性、结合律等）但不具备线性结构，直接优化困难；
+ 李代数（如 so (3)、se (3)）是李群单位元处的切空间，本质是线性向量空间（可做加法、数乘），且定义了李括号运算，能将李群的非线性变换 “局部线性化”；
+ 两者通过指数映射（李代数→李群，如旋转向量→四元数）和对数映射（李群→李代数，如四元数→旋转向量）双向关联，是 SLAM 中处理姿态 / 位姿优化的核心数学工具（将非线性的变换优化转化为线性的向量优化）。
