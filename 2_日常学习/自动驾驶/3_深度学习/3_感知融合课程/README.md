## 自动驾驶中的感知模块

+ 激光雷达感知模块
    + 点云目标检测
    + 点云目标分割
    + 点云路面或者FSD
    + 点云车道线提取
    + 后处理模块
+ 视觉感知模块
    + 2D或3D目标检测
    + TL检测和识别
    + 道路分割或FSD
    + 距离估计
