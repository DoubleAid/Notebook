# <font color="deepskyblue">点云预处理算法</font>

## <font color="skyblue">噪声过滤算法</font>

### <font color="YellowGreen">统计滤波（Statistical Outlier Removal）</font>

核心思想是基于点云局部邻域的统计特性（距离均值、方差），识别并剔除与周围点 “偏离过大” 的离群点

具体逻辑：

1. 对每个点，计算其与邻域内所有点的欧氏距离 $d_i = \sqrt{(x - x_i)^2 + (y - y_i)^2 + (z - z_i)^2}$；
2. 基于这些距离计算统计量（均值 μ、标准差 σ）
3. 计算所有点的$\mu_{\text{global}}$ 和 $\sigma_{\text{global}}$，$\mu_{\text{global}}$ 是所有点的邻域距离均值的全局平均，$\sigma_{\text{global}}$ 是全局标准差，避免单个点的局部统计偏差
4. 若目标点 P 的平均邻域距离 $\mu$ 满足 $\mu > \mu_{\text{global}} + k\sigma_{\text{global}}$，则剔除 P；

k 控制滤波 “严格程度”：k 越小，滤波越严格（剔除更多点，可能误删有效稀疏点）；k 越大，滤波越宽松（保留更多点，可能残留噪声）；

默认值：k=1.0（对应正态分布 68.27% 的置信区间），工业界常用 $k=1.0 \sim 2.0$。

缺点是计算量大

### <font color="YellowGreen">半径滤波（Radius Outlier Removal）</font>

对每个点，统计其搜索半径 r 内的邻域点数，若少于阈值则视为噪声点剔除

计算速度快，适合剔除孤立点

对稀疏点云不友好

### <font color="YellowGreen">条件滤波（Conditional Removal）</font>

基于点的属性（如 XYZ 坐标、强度）设置过滤条件，剔除不符合条件的点

### <font color="YellowGreen">高斯滤波（Gaussian Filter）平滑曲线</font>

高斯滤波（Gaussian Filter）是点云 / 图像领域常用的线性平滑滤波，核心思想是基于高斯函数的权重分布，对目标点及其邻域点进行加权平均，在保留点云整体结构的同时抑制噪声。

设目标点 $P(x_0,y_0,z_0)$，其邻域内有 n 个点 $P_i(x_i,y_i,z_i)$，高斯滤波后新的目标点 $P'(x',y',z')$ 计算如下：

1. 计算每个邻域点 $P_i$ 相对于 P 的相对坐标：$dx_i = x_i - x_0$，$dy_i = y_i - y_0$，$dz_i = z_i - z_0$；
2. 代入 3D 高斯函数，计算每个邻域点的权重 $w_i$:
    $w_i = G(dx_i, dy_i, dz_i) = \frac{1}{(2\pi\sigma^2)^{3/2}} \cdot e^{-\frac{dx_i^2 + dy_i^2 + dz_i^2}{2\sigma^2}}$
3. 对权重进行归一化（避免因邻域点数量不同导致结果偏差）
    $\hat{w}_i = \frac{w_i}{\sum_{i=1}^n w_i}$
4. 加权平均得到新目标点坐标：
    $$
    x^′=∑_{i=1}^nw^i⋅x_i,
    y^′=∑_{i=1}^nw^i⋅y_i,
    z^′=∑_{i=1}^nw^i⋅z_i
    $$​

## <font color="skyblue">降采样算法</font>

### <font color="YellowGreen">体素滤波（Voxel Grid Downsampling）</font>

将 3D 空间划分为固定大小的体素（Voxel），每个体素用「中心点 / 平均点 / 随机点」代表

+ 小尺寸 (0.01 - 0.05)：保持细节，适合高精度场景
+ 中尺寸（0.05 - 0.10）：平衡细节和效率
+ 大尺寸（0.10 - 0.50）：显著减少点数，适合大规模场景

生成代表点的方法

+ 质心法：计算体素内所有点的质心作为代表点
+ 几何中心法：体素立方体的中心点
+ 随机点法：随机在所有点里选一个点

### <font color="YellowGreen">均匀采样（Uniform Sampling）</font>

按固定步长（如 0.05m）在点云中均匀选取点，或随机选取固定数量的点

### <font color="YellowGreen">曲率采样（Curvature-Based Sampling）</font>

计算每个点的曲率，优先保留曲率大的点（如墙角、边缘），曲率小的点（如墙面）降采样

### <font color="YellowGreen">基于半径的下采样（Radius-Based Sampling）</font>

对每个点，若其半径 r 内已有采样点，则跳过该点
