# 矩阵

## 矩阵的奇异性

一个方阵​ A 是奇异矩阵（Singular Matrix）当且仅当：

+ 行列式为零：det(A) = 0
+ 不可逆：不存在逆矩阵 A⁻¹
+ 秩亏缺：rank(A) < n（对于n×n矩阵）

## 正定矩阵

一个矩阵是正定矩阵，必然有以下性质

+ 可逆性: 正定矩阵必然是可逆的，并且他的逆矩阵也是正定矩阵
+ 行列式：正定矩阵的行列式大于0
+ 主元：在矩阵进行LU分解（高斯消元）时，所有的主元（pivots）都是正的
+ Cholesky 分解：正定矩阵A可以唯一的分解成 $A = LL^T$ 的形式，其中 L 是一个下三角矩阵。这个分解在数值计算中极其重要，比一般的LU分解更快更稳定
+ 与单位矩阵合同：存在可逆矩阵 P， 使得 $A = PP^T$ 。 这表明正定矩阵可以看作是单位矩阵经过基础变换得到的

其他性质

+ 通常我们讨论的正定矩阵默认是实对称矩阵。
+ 二次型条件：对于所有非零向量 x ∈ Rⁿ，都有：$xᵀ A x > 0$
+ 主子式条件：矩阵 A 的所有顺序主子式均为正数。顺序主子式：左上角的 1×1 子矩阵的行列式，2×2 子矩阵的行列式，...，直到 n×n 矩阵 A 本身的行列式。
