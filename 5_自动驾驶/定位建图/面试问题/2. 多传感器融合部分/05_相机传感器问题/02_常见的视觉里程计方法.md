# 常见的视觉里程计

## 1. 特征点法

提取、匹配图像间的显著点，建立2D-2D或3D-2D对应关系，通过对极几何或PnP求解相机运动。

流程：特征提取 → 特征匹配 → 误匹配剔除 → 运动估计 → 局部优化

代表：ORB-SLAM系列（经典）、VINS-Mono（融合IMU）

## 2. 直接法

不提取特征点，而是直接最小化像素亮度误差（光度一致性假设）来估计运动。

流程：像素块选取 → 光度误差构建 → 优化求解运动 → 关键帧选择

代表：DSO、LSD-SLAM

## 3. 多传感器融合

结合视觉的高精度和 IMU 的高频特性，解决纯视觉的尺度漂移、遮挡失效等问题，是车载 / 无人机 SLAM 的主流方案。

基于光流追踪的特征点法 VINS，ORB-SLAM3

## 4. 基于模型的方法

数据驱动的端到端方案，利用 CNN+RNN 网络直接回归相机位姿或深度，无需手动设计几何约束。

DeepVO
