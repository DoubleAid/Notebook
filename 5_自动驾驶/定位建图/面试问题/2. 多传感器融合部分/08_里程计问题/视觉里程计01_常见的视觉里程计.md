# 常见的视觉里程计

常见的视觉里程计（VO）可按核心匹配方式分为直接法，特征法和半直接法

## 特征点法视觉历程计（最经典，鲁棒性强）

1. 经典方案：ORB-SLAM 系列（ORB-SLAM/2/3）
    + 核心：提取ORB特征点
    + ORB-SLAM3 新增 IMU 紧耦合和多地图融合，是视觉 SLAM 的标杆方案；
    + 适用场景：室内外通用场景，对光照变化、运动模糊有一定鲁棒性。

## 直接法视觉里程计（无特征提取，高精度）

核心逻辑：直接最小化两帧图像的像素灰度差，无需提取特征，对弱纹理场景友好。

1. VINS-MONO

## 半直接发视觉里程计

核心逻辑：提取稀疏特征点，用直接法优化像素灰度，兼顾速度、精度和鲁棒性。

1. DSO 的优化版：Direct Sparse Odometry with IMU（DSO+IMU）
    + 核心：在 DSO 基础上融合 IMU 预积分，抑制运动畸变，解决直接法对快速运动的鲁棒性问题；
    + 特点：适合高速运动场景（如无人机飞行）。

## 视觉惯性里程计（VIO 工业主流方案）

融合视觉 + IMU，弥补纯视觉的短板（如快速运动、光照失效），是自动驾驶、无人机的核心定位模块。

1. MSCKF（Multi-State Constraint Kalman Filter）
    + 核心：基于 ESKF 框架，IMU 做预测，视觉构建多相机状态约束，不估计特征点深度，计算量小、实时性强；
    + 特点：适合高实时性场景，是车载 VIO 的经典方案。
2. OKVIS/OpenVINS
    + 核心：基于关键帧的非线性优化 VIO，紧耦合视觉与 IMU，支持单目 / 双目，开源易用；
    + 特点：精度高，适合学术研究和中小型项目落地。
3. 工业级方案：VINS-Mono/VINS-Fusion
    + 核心：单目 VIO 标杆方案，通过滑动窗口优化处理视觉与 IMU 约束，VINS-Fusion 支持多传感器融合（视觉 + IMU+GPS）；
    + 特点：鲁棒性强，适合无人机、机器人等场景，是国内高校和企业广泛使用的方案。

## 面试总结

+ 纯视觉里程计优先提 ORB-SLAM 系列（特征法）、DSO（直接法），区分两者核心差异；
+ 工业落地必提 VIO 方案（MSCKF、VINS-Mono），强调 “视觉 + IMU” 融合的优势（抗干扰、无漂移）；
+ 按场景选型：弱纹理选直接法，通用场景选特征法，高实时性选 MSCKF，高精度选 VINS-Mono。
