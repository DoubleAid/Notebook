# 常见的激光里程计

## 一句话回答

常见的激光里程计可按核心匹配方式分为直接法，特征法和混合法

### 直接点云匹配（无需特征提取，精度高）

1. ICP（迭代临近点）及其变种
    + 直接最小化相邻点云的对应点距离，迭代求解位姿变换
    + 变种：GICP（广义ICP）：考虑点云法向量和协方差，对旋转更鲁棒，适合低分辨率激光雷达；
2. 变种 NDT（正态分布变换）：将点云划分为网格并拟合正态分布，通过分布相似度匹配，速度比 ICP 快，适合大规模场景（如自动驾驶）。

3. LOAM系列直接匹配分支
    + 代表：LeGo-LOAM（地面优化 + 特征提取结合），Fast-LIO2 (激光 + IMU紧耦合，直接匹配点云)
    + 核心：融合IMU预积分约束，降低运动畸变影响，提升动态场景鲁棒性

### 特征提取匹配（速度快，鲁棒性强）

1. LOAM（激光里程计和建图）

    + 核心：提取点云中的边缘特征和平面特征，分别匹配相邻帧特征，求解位姿
    + 特点：降低计算量，对非结构化环境适应性好，无回环检测，需要搭配后端优化

2. Cartographer-Lidar

+ 核心：基于分支定界法的实时子图匹配，将激光帧匹配到局部子图，而非相邻帧。
+ 特点：支持多激光雷达融合，通过子图累积减少误差，适合室内大场景建图。

### 多传感器融合激光里程计（工业界）

1. 激光 + IMU 融合

    + 代表：FAST-LIO2/2, LIO-SAM, LIOM
    + 核心：IMU 预积分提供运动先验，激光点云做观测修正，抑制激光匹配的运动畸变，支持高速、剧烈运动场景（如无人机、自动驾驶车辆）。

2. 激光 + 视觉 + IMU

    + 代表：LVIO-SAM, VIL-SAM
    + 核心：视觉提供纹理特征，激光提供几何特征，IMU 弥补视觉失效场景（如强光、无纹理），实现全天候、全场景定位。
