# 常见的激光雷达和类型和参数

## 常见的激光雷达类型

### 机械式激光雷达

+ 核心原理：通过电机带动发射 / 接收模组旋转，实现 360° 全向扫描，线束覆盖垂直视场角
+ 代表产品：禾赛 Pandar128、速腾聚创 RS-Ruby、Velodyne HDL-64E
+ 优势：线束多（16/32/64/128 线）、点云密度高、垂直分辨率优、成熟稳定
+ 应用场景：自动驾驶乘用车、高精地图采集车、固定场景建图

### 半固态激光雷达（混合固态）

+ 核心原理：简化机械结构，通过微振镜 / 棱镜偏转光束，无需 360° 旋转，扫描视场角可定制
+ 代表产品：禾赛 AT128、速腾聚创 M1、图达通 Falcon
+ 优势：成本低于机械式、体积小、可靠性高、点云密度均衡
+ 应用场景：自动驾驶前装量产车、机器人导航、园区无人车

### 纯固态激光雷达（Flash）

+ 核心原理：无任何运动部件，通过面阵发射器一次性发射大范围激光，接收阵列同步成像
+ 代表产品：大疆 Livox Horizon（MEMS）、法雷奥 Scala
+ 优势：体积最小、抗振动、成本低、刷新率高
+ 应用场景：低速无人车、机器人避障、车内感知

补充：按线束数量划分（行业常用口径）

这是工程师选型时的直观维度，线束数量直接决定垂直分辨率：

+ 低线束（4/8/16 线）：成本低、点云密度低，适合低速机器人、室内建图；
+ 中线束（32/64 线）：性价比高，兼顾距离与密度，适合园区无人车、高精地图采集；
+ 高线束（128/256 线）：点云密度极高，垂直分辨率＜0.1°，适合高速自动驾驶、复杂城市道路感知。

## 核心参数

+ 测距性能：
  + 最大探测距离：指对10%反射率目标的探测距离
  + 测距精度：测量距离的误差范围，通常为 +/- 2cm - 5cm
+ 扫描性能
  + 线束数量：垂直方向的激光发射通道数，线束越多，垂直分辨率越高
  + 水平视场角 FOV: 机械式雷达通常为360°，半固态雷达多为 120°~150°
  + 水平分辨率: 相邻两个激光点的水平夹角，水平分辨率=fov/扫描一行的点数
  + 刷新率：每秒完成的扫描次数,单位是Hz，通常为10Hz
+ 点云密度：points/m^2
