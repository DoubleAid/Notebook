# 基于给定的空间三角形，写一个可以在该三角形中生成随机数的函数

## 方法一：标准方法（重心坐标法）--- 正确且均匀

这是最常用且正确的方法，可以保证点在三角形内均匀分布

想象一下三角形ABC内部有一个点P。我们可以用点P将三角形划分成三个小三角形：$\Delta{PBC}, \Delta{PCA}, \Delta{PAB},$

**核心思想**：点P在三角形ABC内的充要条件是，这三个小三角形的面积之和等于原三角形ABC的面积，并且每个小三角形的面积都非负。

更精确地说，点P的重心坐标 (u, v, w)实际上就是这三个小三角形面积与原三角形面积的比值：

$$
u = 面积(\Delta{PBC}) / 面积(\Delta{ABC})
v = 面积(\Delta{PCA}) / 面积(\Delta{ABC})
w = 面积(\Delta{PAB}) / 面积(\Delta{ABC})
$$

由于 u + v + w = 1, 点 P 在三角形内的条件就是：u >= 0, v >= 0, w >= 0

如果任何一个坐标为负，说明点P在三角形的外部

所以，随机生成 u, v, w, 再计算 P 点，这样就可以生成内部点
