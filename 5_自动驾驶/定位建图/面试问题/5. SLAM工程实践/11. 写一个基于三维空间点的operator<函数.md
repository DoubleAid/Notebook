# 写一个基于三维空间点的operator<函数

主要区别是 判断的方式，主要是以下几种:

+ 字典序排序: 先比较x，再比较y，最后比较z
+ 基于距离原点的长度排序
+ 自定义排序

## 字典序排序

```cpp
#include <tuple>

struct Point3D {
    double x, y, z;

    bool operator<(const Point3D& other) const {
        return std::tie(x, y, z) < std::tie(other.x, other.y, other.z);
    }
};
```

## 基于距离原点的长度排序

```cpp
#include <cmath>

struct Point3D {
    double x, y, z;

    bool operator<(const Point3D& other) const {
        double len1 = x*x + y*y + z*z;
        double len2 = other.x*other.x + other.y*other.y + other.z*other.z;
        return len1 < len2;
    }
};
```
