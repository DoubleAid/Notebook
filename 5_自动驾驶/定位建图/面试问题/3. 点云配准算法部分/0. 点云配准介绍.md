# 点云配准介绍

## 什么是点云配准

从技术上讲，点云配准就是求解一个刚体变换（即只包含旋转和平移，不改变物体形状大小的变换），使得两个点云之间的差异最小化。

这个刚体变换可以用一个变换矩阵 T​ 来表示（通常是4x4矩阵）。
给定两个点云：

+ 源点云：$P = {p_i}$（待移动的点云）
+ 目标点云：$Q = {q_j}$（作为参考基准的点云）

点云配准的目标是找到一个最优的变换矩阵 T，使得将 T应用于源点云 P后，得到的新点云 T(P)与目标点云 Q尽可能重合。
用数学公式表示就是求解：
$$
T* = argmin_T\sum{dist( T(p_i), Q )}
$$
其中 dist是距离度量函数（如最近点距离）
