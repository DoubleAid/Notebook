# 点云配准的三大假设

## 假设一：点云之间存在"重叠区域"

待配准的两片点云（源点云和目标点云）必须有一部分共同描述了**同一个物理表面**。重叠区域的大小和质量直接决定了配准是否可能成功

+ **为什么重要**：配准算法的本质是在目标点云中为源点云的点寻找正确的"对应点"。如果两片点云完全没有重叠，它们描述的是完全不同的物体或场景，那么寻找正确的空间变换就失去了意义
+ **实践意义**
  + 在进行了粗配准（如使用特征匹配）后，通常会先基于包围盒或其他方法快速估计一个大致的重叠区域，并只在重叠区域内进行精细配准，这可以大大提高效率和准确率
  + 当配准效果不佳时，首先需要检查的就是点云的重叠度是否足够

## 假设二：点云之间的几何变换是 "刚体变换"

假设在两次扫描之间，点云所描述的物体或场景本身没有发生形变。连接两片点云的空间变换是一个刚性变换，即只包含旋转和平移。变换前后，点云中任意两点之间的距离和角度保持不变

+ **为什么重要**：这是经典ICP算法及其大部分变种的基础。算法求解的目标就是旋转矩阵 R 和一个平移向量 t，使得变换后的源点云和目标点云对齐。这个假设极大的简化了问题的复杂度
+ **实践意义**
  + 该假设非常适合刚体物体的配准，如工业零件，建筑物，静态场景等
  + 对于非刚性物体的配准（如人体，衣物，软组织），这个假设不成立，需要使用更复杂的非刚性配准算法

## 假设三：点云中的噪声和异常值是"可处理的"

承认点云数据中存在测量噪声（如传感器误差），遮挡造成的确实以及异常值（如漂浮点），但假设这些干扰的水平和影响是有限的，并且可以通过算法策略（如鲁棒损失函数，外点剔除）来减弱负面影响，从而仍能够估计出正确的变换关系

+ **实践意义**
  + 原始ICP使用最小二乘误差，对异常值非常敏感。因此，发展出了多种鲁棒ICP变种：
  + Point-to-Plane ICP: 通过将点匹配到切平面，对沿表面方向的噪声更鲁棒
  + Trimmed ICP: 在每次迭代中舍弃掉距离最远的那部分对应点（认为是异常值）
  + 使用鲁棒核函数的ICP：如Huber损失函数，减少大误差对整体目标函数的过度影响
