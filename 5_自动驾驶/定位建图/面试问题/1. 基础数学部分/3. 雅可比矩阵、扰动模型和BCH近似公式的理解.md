# 雅可比矩阵，扰动模型 和 BCH 近似公式

## 三者的介绍

+ 雅可比矩阵：描述函数（如 SLAM 中残差）对输入变量（如位姿）的局部线性变化关系，是非线性优化中求导、更新参数的核心工具；
+ 扰动模型：给李群变换加微小扰动（对应李代数向量），通过雅可比将扰动对结果的影响线性化，避免直接对李群求导的复杂性；
+ BCH 近似：解决李群乘法的非线性问题，将两个李群元素的乘积（或指数映射的和）近似为李代数的线性组合，简化多变换叠加时的优化计算。

## 三者的关系

+ 雅可比是线性化的核心工具，扰动模型是李群场景下的「求导方法论」，BCH 近似是扰动模型的「数学补充」；
+ 逻辑链：SLAM 优化需对李群位姿求导→直接求导复杂，用扰动模型给位姿加微小扰动（转化为李代数向量）→扰动叠加涉及李群乘法，用 BCH 近似简化计算→最终通过雅可比矩阵量化扰动对残差的影响，实现线性优化。

## 为什么使用扰动模型

+ 李群不支持线性运算，直接对其求导（如残差对位姿的导数）不成立，扰动模型将位姿优化转化为李代数向量的线性优化，适配梯度下降等算法；
+ 避免直接操作李群导致的非法变换（如优化旋转矩阵后不再满足正交性），扰动是微小量，通过指数映射回李群时天然保证变换合法性；
+ 简化求导计算：扰动模型将复杂的李群求导转化为李代数向量的普通求导，结合 BCH 近似可快速推导雅可比表达式，提升优化效率。
