## <font color="coral">Node 是什么</font>
在 Kubernetes 中， Node 通常是指实体机或者虚拟机，一个 node 可以指 AWS 的一台 EC2 或者 GCP上的一台 computer engine，或者是实际中的计算机， 只要上面装有 docker engine， 足以跑起 pod， 就可以加入 kubernetes cluster
